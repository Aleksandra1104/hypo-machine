**Java HYPO machine + OS simulator**

-- Absolute loader for HYPO executable files
-- CPU instruction execution with registers + stack support
-- Dynamic memory management using OS/User free lists (allocate/free + block merging)
-- Process management with PCBs, Ready Queue / Waiting Queue, dispatch/save context
-- Priority-based scheduling with time-slicing
-- Interrupt handling and system calls (I/O completion, alloc/free memory, clock get/set)
-- Memory dumps to console and output log file

Note: The **hw4** branch contains the final, most complete version of this project.
